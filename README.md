## Featured Project

### GoFix — A Smart App for Home Services (Deployed On Play Store)
*Final Year Project · Bachelor of Science in Computer Science*

GoFix is an AI-powered, Flutter and Firebase-based marketplace platform that connects customers with verified home service technicians (electricians, plumbers, AC repair specialists, and others) through a structured, transparent digital workflow.

**Key Contributions and Features:**
- Designed and implemented GPS-based, location-aware technician matching
- Built a competitive bidding system enabling technicians to submit offers and customers to select the best fit
- Developed real-time job tracking with live status updates between customers and technicians
- Implemented in-app chat and direct calling for seamless communication
- Integrated an AI-powered smart price estimation feature to support fair cost agreements
- Built an AI assistant for technicians, providing instant guidance for on-the-job issues
- Developed an administrative dashboard for technician verification, complaint resolution, and system monitoring

**Technology Stack:** Flutter · Firebase (Firestore, Authentication, Cloud Messaging) · Android Studio
